# 家計簿アプリ

## どんなアプリ
* グループの家計状況を管理できるアプリ
* ユーザはグループに所属し、毎日の出費などを記録できる  
* 家計の状況は常にダッシュボード上で確認でき、  
  閾値を設定することでユーザに家計状況をお知らせできる

## 技術要件
### フロントエンド
  * Vue.js 2 (TypeScript)
    * Vuex
    * Vue Router
  
### バックエンド
  * SpringBoot (Java8)
    * Spring MVC
    * Spring Data JPA
    * Spring Security
